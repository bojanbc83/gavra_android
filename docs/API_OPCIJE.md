# API Opcije za Realtime ETA

## OpenRouteService

**1. Public API (openrouteservice.org)**
- 2000 zahteva dnevno besplatno
- Potrebna registracija za API key

**2. Self-hosting (laptop/cloud)**
- Neograničeno
- Docker instalacija

---

## GraphHopper

**1. Public API (graphhopper.com)**
- Free plan: samo nekomercijalnu upotrebu, ograničeno
- Basic: 69€/mes - 5000 kredita/dan
- Standard: 199€/mes - 15000 kredita/dan

**2. Self-hosting (laptop/cloud)**
- GraphHopper Routing Engine je open source
- Može se hostovati na laptopu/cloud-u besplatno i neograničeno

---

## Valhalla

**1. Demo server (valhalla.openstreetmap.de)**
- Besplatno, fair use, rate limited

**2. Self-hosting (laptop/cloud)**
- Neograničeno
- Docker, Python (`pyvalhalla`), NodeJS

---

## HERE

**Samo cloud API (nema self-hosting)**
- Besplatni tier: potrebna registracija
- Link: https://developer.here.com
- Napomena: pricing se menjao, proveriti aktuelne uslove
